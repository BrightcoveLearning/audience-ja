    ブライトコーブオーディエンスのよくある質問

    このトピックでは、ブライトコーブオーディエンスに関してよく寄せられる質問のいくつかについて説明します。

    オーディエンスの理由とその仕組み

    マーケティングオートメーションソリューションで動画を追跡する必要があるのはなぜですか？
    マーケティング自動化ソリューションは、今日の動画を追跡しません。強力なコンテンツとして動画に投資する場合は、その効果を追跡し、コンテンツを閲覧したリード/ユーザーとのコミュニケーションを自動化する必要があります。
    ブライトコーブとオーディエンスモジュールはクッキーを使用していますか？
    Audience モジュールは、統合が行われるときにユーザーを識別するために、サードパーティ Cookie を作成して使用します。この方法では、視聴者は 2 回目にリードフォームへの記入を求められることはありません。このモジュールでは、Marketo、HubSpot、Eloqua API も利用し、クッキーを使用してビューアを追跡します。オーディエンスプロファイルを使用する場合、ブライトコーブ Cookie が使用されます。
    ブライトコーブオーディエンスはすべての顧客に対して有効になっていますか？
    なし視聴者は、Video Marketing Suite（VMS）の購入可能なアドオンとして利用できます。ただし、オーディエンスプロファイルは、すべてのブライトコーブのお客様が無料でご利用いただけます。あなたはそれを「Audience-Lite」と考えることができます。
    オーディエンスプロファイルの対象となるのは誰です。
    2017 年第 4 四半期以降、すべての VM のお客様は無料でオーディエンスプロファイルにアクセスできます。トラッキングデータを開始するには、この機能をアクティブ化する必要があります。エンタープライズビデオスイート（EVS）のお客様は、オーディエンス SSO トラッキングも利用できます。

    視聴者の追跡と制限

    フル機能のオーディエンスモジュールと比較して、オーディエンスプロファイルにはどのような制限がありますか？
    • サードパーティ製システムとの統合は不要
    • APIアクセスなし
    • カスタムリードフォームなし (オーディエンスリードフォームのみ)
    • 1000 ビュー/最大 24 時間キャプチャ時間
    ただし、オーディエンスデータを CSV または XLS ファイルに手動でエクスポートできます。
    マーケティング自動化プラットフォームに接続すると、オーディエンスはどのような制限/要件がありますか？
    制限と要件については、各プラットフォームのドキュメントに記載されています。
    • オラクル・エロクア
    • ハブスポット ( クライアントサイド API または REST API )
    • Marketo ( マンチキンまたはREST API )
    • セールスフォース
    オーディエンス API をポーリングできる頻度に制限はありますか？動画検索結果に1000のキャップがあることは知っていますが、そのAPIエンドポイントには他の周波数関連のキャップがありますか？
    API 呼び出しあたりのビューイベントの最大数は 100 です。レート制限はありませんが、あまりにも多くを呼び出そうとすると、おそらく少し遅くなります。
    現在発生している 3 時間の遅延よりも早くオーディエンスデータを取得する方法はありますか？私は「リアルタイムに近い」指標を取得したいと思います。
    なしリアルタイムメトリクスが必要な場合は、独自のプラグインを作成し、データを自分のシステムに送信する必要があります。
    イベントビューがオーディエンスに送信されるトリガーは何ですか？ビデオが完了したとき、プレーヤーウィンドウが閉じられたとき、または何か他のものですか？ユーザーがビデオを完了せず、プレーヤーウィンドウを閉じない場合はどうなりますか？
    これは様々ですが、基本的に、Audience プラグインは 10 秒ごとにハートビートをサーバーに送り返します。
    視聴者がビューのトラッキングを「完了」と見なすのはいつですかプレーヤーウィンドウが閉じているときですか、ビデオが完成したときですか？彼らが間近で見て、ビデオを一時停止し、プレーヤーウィンドウを閉じない場合はどうなりますか？
    私たちが最後にハートビートを見たから15分が経過した時です。基本的に活動が不足しています。
    オーディエンスの閲覧データで使用可能になった視聴者データを変更することは可能ですか？そのイベントデータは、将来何らかのログ統合によって更新されますか？
    UI または API 経由でオーディエンスにデータが使用可能になると、データは「決済済」とみなされ、変更されません。

    データ保護と記録保持

    オーディエンス GDPR に準拠していますか？

    はい。オーディエンスは、オーディエンスを介して収集したデータを検索して削除し、ユーザーの代わりに操作することができます。

    https://audience.support.brightcove.com/admin/managing-viewer-data.html

    オーディエンスにデータを保持できる期間はどれくらいですか？
    MAP 統合は最大 6 か月に設定でき、非 Map 統合は 3 年に設定できます。
    GUID ルックアップがキャッシュされる期間はどれくらいですか
    既知の場合は 30 日、不明の場合は 7 日間 (MAP 統合にのみ適用され、非 Map 統合にはルックアップは必要ありません)

    マーケティング自動化とビューア分析

    マーケティングオートメーションプラットフォームの他のメディアと動画のトラッキングについて何が違うのですか？
    ウェブページやダウンロード可能なホワイトペーパーとは異なり、ビデオを使用すると、視聴者が視聴したビデオの量、つまりどのコンテンツが公開されているかを正確に判断できます。
    Eloqua/marketo/HubSpotアカウントとどのように接続しますか？
    Eloqua/marketo/HubSpot アカウントには、オーディエンスモジュールの Video マーケティングスイートから直接ログインできます。そこから、MAPトラッキングを内蔵したプレイヤーとリードフォームを作成できるため、動画分析のトラッキングをすぐに開始できます。
    Eloqua/marketo/HubSpotページだけでなく、非Eloqua/marketo/HubSpotのランディングページ（サイトコア/ワードプレス/ギャラリーなど）のビューを追跡できますか？
    はい。使用されているプレーヤーが「オーディエンス」が有効になっている（オーディエンスプラグインがインストールされている）限り、プレイヤーが MAP ランディングページかその他のランディングページにあるかにかかわらず、すべてのビューが追跡されます。
    この統合で追跡できるものは何ですか？
    当社の統合により、データベースにすでに存在する既知の連絡先の動画視聴分析（動画、動画 ID、閲覧率、視聴時間）を追跡できます。
    不明な連絡先はどうですか？彼らも追跡できますか？
    ブライトコーブオーディエンスを使用すると、不明な連絡先を追跡できます。その後、Eloqua/HubSpot フォームを送信すると（既知になる）、オーディエンスは視聴率の情報を新しい連絡先に遡及的に関連付けることができます。視聴者は毎日視聴率情報を再送信し、不明な連絡先からのビューデータをOracle Eloqua/HubSpotインスタンスに照合します。
    私のビデオビューのデータは、マップ内のどこに表示されますか？
    これは、使用している MAP によって異なります。Oracle Eloqua 、 Marketo 、 HubSpotについては、次のドキュメントを参照してください。
    Salesforce の動画視聴データはどこに表示されますか？
    ビデオビューは、[アクティビティ] セクションのビューアの連絡先の下にタスクとして Salesforce に配信されます。この例については、「 Salesforce でのビデオデータの表示」を参照してください。
    連絡先の 1 つが動画を視聴したときに Salesforce に通知を受けることはできますか？
    直接じゃないただし、連絡先の活動を通知するレポートを実行することはできます。
    SalesforceとOracle Eloqua/Marketoとの追跡に違いはありますか？
    最大の違いは、Salesforce がユーザを追跡するために Cookie を使用しないことです。したがって、ビューアを識別するために、一意の識別子を渡す必要があります。詳細については、このドキュメントを参照してください。さらに、CRM である Salesforce は、自動化されたフォローアップやリードスコアリングを提供していません。ただし、Salesforce で ToDo を作成し、実際のフォローアップのために営業チームに割り当てることができます。
    オーディエンスには「カスタム」統合オプションがあり、これを使用してオーディエンスが統合できる他のプラットフォームは何ですか？

    視聴者は、次のようなデータプラットフォームと「統合」できます。

    1. 各連絡先レコードに一意の識別子を使用します。
    2. サードパーティプラットフォーム（オーディエンスなど）から追加データを受信できる API がある

    プレイヤーリードフォーム

    オーディエンスにリードフォームが表示されるのはいつですか
    デフォルト設定では、MAP データベース内の不明なユーザー (既知のユーザーに変換するユーザー) へのリードフォームのみが表示されます。リードフォームを常に表示するように選択した場合は、設定でこれを上書きできます。これはテストに最も役立ちます。既知のユーザーは自動的に追跡され、そのデータは MAP 内の連絡先に関連付けられ、フォームを表示する必要はありません。
    リードフォームをカスタマイズできますか？
    UI を使用して、表示されるテキストとフィールドを調整できます。UI で利用可能なものを超えて変更することはできません (ギャラリーでは、フォームがサイトと一致するようにスタイルを更新できます)

    一般的なトピック

    どの埋め込みコードを使うべきですか？iframeかページ内？
    オーディエンスが機能するためには、高度な埋め込みコードを使用することが必要です。標準（iframe）埋め込みコードを使用すると、プレーヤーはページから分離されます。つまり、オーディエンスプラグインは一意の識別子を見ることができないため、すべてのユーザーは「不明」として表示されます。
    マーケティング自動化とは何ですか？
    マーケティング自動化は、さまざまなオンラインマーケティングチャネル（電子メール、ウェブ、フォーム、コンテンツ、ビデオ、ソーシャルなど）とのリード/ユーザーのインタラクションを追跡する技術です。マーケティング自動化により、マーケティング担当者はこれらのチャネル全体でより効果的にマーケティングを行い、反復的なマーケティングタスク（電子メール通信など）を自動化できます。マーケティング自動化により、マーケティング担当者は事前定義された変数に基づいて、自動化されたパーソナライズされたタスクを設定できます。たとえば、ユーザーが特定の動画の 75% を視聴した場合、その特定の件名に関するメールを送信しますが、12% しか視聴していない場合は別のものを送信します。
    ブライトコーブはマーケティングオートメーション企業ですか？
    いいえ、ブライトコーブは動画マーケティングのための完全なソリューションを提供しています。当社のソリューションにより、動画をオンラインで入手し、コンテンツを整理、プレーヤーの作成、ビデオポータルの作成、分析の追跡、既存のマーケティング自動化プラットフォームとの統合を行うことができます。ブライトコーブは、動画分析をマーケティングオートメーションプラットフォームに統合します。
    MarketOカスタムアクティビティが正しく設定されているかどうかを確認するにはどうすればよいですか？
    オーディエンスは、統合の特定の種類のためのいくつかのデバッグツールを提供する隠された URL を持っています, Marketo REST は 1 つです. https://studio.brightcove.com/products/videocloud/audience/debug
    カスタムインテグレーションを構成するには何が必要ですか？
    3つのことを行う必要があります：
    1. オーディエンスがキャプチャできるページの URL またはコードに一意の識別子を公開する必要があります。
    2. Audience API を使用してデータを JSON ファイルとして取得するスクリプトを作成します（Brightcove グローバルサービスがこれを支援します）。
    3. データを宛先データベースのフィールドにマップします (スプレッドシートの結合を考えて)。
    次のサポートドキュメントでは、これらのタスクについて説明します。
    • オーディエンスモジュールを使用したカスタムインテグレーションの作成-オーディエンスがキャプチャできる一意の識別子の提供方法を説明します
    • Audience API -プログラムですべてのデータを JSON ファイルとして取得し、選択したデータ処理システムにインポートする方法について説明しています
    オーディエンスはCDOデータをEloquaにプッシュするにはどうすればよいですか？
    カスタムオブジェクトバルク API を使用します。https://docs.oracle.com/cloud/latest/marketingcs_gs/OMCAC/api-bulk-2.0-customobjects.html
    オーディエンスは外部資産活動をどのようにプッシュするのですか？「キャンペーン」、「外部資産」および「活動タイプ」を使用する方法を制御できますか。
    外部アクティビティ API（ https://docs.oracle.com/cloud/latest/marketingcs_gs/OMCAC/op-api-rest-2.0-data-activity-post.html ）を使用しています。キャンペーンの名前、アセット名、タイプを制御することはできません。私たちは、独自のキャンペーン（BrightCoveVideoPlay という名前の）を作成し、アクティビティの作成に進みます。BrightCoveVideoPlay という名前もあります。
    オーディエンスがサービス X と通信できるように、ホワイトリストに登録できるIPアドレスは何ですか？
    オーディエンスは、IP ホワイトリストに使用する一連の IP アドレスを確立しました。リストを表示するには、Eloqua 接続を編集し、 Eloqua でオーディエンスデータが表示されませんか？リンクをクリックして、IP アドレスの一覧を表示します。
