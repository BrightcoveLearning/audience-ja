    Oracle Eloqua用のカスタムリードフォームを作成する

    このトピックでは、Oracle Eloqua フォームを使用して Brightcove Player 内でリードデータをキャプチャする方法について説明します。

    Audienceモジュールではリードフォームを作成できます。リードフォームを使用すれば、Brightcove Player内部で動画が再生されると、視聴者情報を取り込むことができます。（表示または非表示の）必要なフィールドを追加し、（利用可能な場合）プログレッシブプロファイリングなどの機能を利用できます。リードデータはEloquaに送信され、Eloquaによって処理されます。

    カスタムリードフォームを作成するには、次の手順を行います。

    1. Eloqua でフォームを作成する
    2. フォームをコピーする HTML
    3. フォームのスタイルを設定する
    4. オーディエンスモジュールでカスタムリードフォームを作成する
    5. リードフォームをプレーヤーに割り当てる
    6. Eloqua でフォームデータを検証する
    注：視聴者の情報をキャプチャし、Video Cloud にデータを保存する標準的なリードフォームについては、「オーディエンスリードフォームの作成」を参照してください。

    Eloquaでフォームを作成する

    Eloquaでは、視聴者情報を取り込むフォームを作成できます。これらのフォームは、ブライトコーブプレーヤーの内部に表示できます。取り込んだデータは、将来のコミュニケーションをカスタマイズする際に使用できます。下の手順では、姓、名、電子メールアドレスを取り込む簡単なフォームを作成します。Eloquaフォームについての詳細は Eloquaのドキュメントを参照してください。

    1. お客様のEloquaアカウントにサインインします。
    2. [Assets] > [Forms]に移動し、[Create a Form]をクリックします。フォームテンプレート選択ウィンドウが開きます。
    3. [Blank Form]をクリックし、[Choose]をクリックします。
    4. 左側のナビゲーションで、[ フィールドグループ ] パネルを展開します。
    5. [ フィールドグループ ] パネルで、[ 基本的な連絡先情報 ] をクリックし、フォームテンプレートにドラッグします。
    6. フィールドがフォームに追加されていることを確認します。
    7. デフォルトでは、フィールドはフィールドマージを使用して追加されます。これが原因で、Audienceで使用すると、奇妙なテキストがフィールドに表示されます。フィールドマージを削除するには、[First Name]フィールドをクリックして選択します。
    8. 左側のナビゲーションパネルで、[ 一般プロパティ ] セクションを探し、[ フィールドデータの事前設定 ] プロパティを [ なし] に設定します。
    9. [Last Name]フィールドと[Email Address]フィールドにも、このプロセスを繰り返します。
    10. [保存]をクリックします。
    11. フォームに名前を付けて、[ 保存 ] をクリックします。

    フォームHTMLをコピーする

    Audienceモジュール内部でフォームを使用するには、フォームHTMLをコピーする必要があります。以下の手順に従って、フォームHTMLをコピーします。

    1. [ アクション] [フォーム HTML > を表示] をクリックします。フォーム HTML が表示されます。
    2. HTML全体をクリップボードにコピーします。[Close]をクリックします。
    3. フォームHTMLを、お気に入りのHTMLエディターに貼り付けます。エディターのコード整形機能を使用して、読みやすいようにコードを整形したい場合があります。

    フォームをデザインする

    デフォルトでは、フォームが Brightcove Player 内に表示される場合、フォームはプレーヤーの上に透明な背景に表示されます。CSSを使用してフォームの基本的なスタイルを設定したいと思う場合があるかもしれません。そのために、下のセクションでフォームをデザインする方法の例を2つ示します。また、関連するCSSも提供します。その際、標準色に適合するよう、または追加フィールドのスタイルを処理するため、CSSの変更が必要になる場合があります。

    簡単なフォームのデザイン例

    この例ではCSSを使用して、フォームの背景色を灰色に変更します。フォームは以下のようになります。

    このスタイル設定では、フォームHTML style タグの上部に以下のCSSを追加します。

    ボディ {
      背景色:rgba (204,204,204,0.9);
      マージン:0;
      パディング:20px;
      高さ:100%;
}

    Audienceモジュールでカスタムリードフォームを作成する

    フォームHTMLをEloquaで作成し、必要に応じてデザインした後、Audienceモジュールでカスタムリードフォームを作成し、リードフォームをAudience対応プレーヤーに関連付けます。

    1. Audienceモジュールを開きます。
    2. 左側のナビゲーションで [ リードフォーム ] をクリックします。リードフォームが作成されている場合、作成されたリードフォームが表示されます。
    3. [リードフォームの作成]をクリックします。
    4. リードフォームに名前を付けます。
    5. フォームの言語を選択します。
    6. フォームのタイミングを選択します。フォームのタイミングで、リードフォームがいつ表示されるかが決まります。
    7. フォームのタイプを[カスタムリードフォーム]に設定します。
    8. フォーム HTML に貼り付けます。
    9. （オプション）確認 HTML に貼り付けます。確認ビューが5秒間表示され、その後、動画の再生が再開されます。確認HTMLが含まれない場合は、フォームの送信後に動画の再生が再開されます。
    10. （オプション）フォームのプレビューを開くには、[フォームのプレビュー]をクリックします。
    11. [保存]をクリックして、リードフォームを保存します。

    リードフォームをプレーヤーに割り当てる

    作成したリードフォームは、プレーヤーに割り当てる必要があります。リードフォームをプレーヤーに割り当てるには、次の手順を行います。

    1. 左側のナビゲーションで [ オーディエンスプレイヤー ] をクリックします。
    2. プレーヤーを見つけ、[ Lead Form ] ドロップダウンリストを使用して、リードフォームを選択します。

    これでセットアップは完了しました。リード情報を取得する準備ができました。Audience対応プレーヤーを使用して動画が公開されると、フォームタイミング設定に従ってリードフォームが表示されます。

    Eloquaでフォームデータを確認する

    フォームデータがEloquaに渡されたことを確認する方法は複数あります。以下の手順に従って、Eloquaでフォームデータを表示します。

    1. お客様のEloquaアカウントにサインインします。
    2. [Assets] > [Forms]に移動します。
    3. 左ナビゲーションで[All Files]リンクをクリックし、フォームをすべて表示します。
    4. フォーム名をダブルクリックして開きます。
    5. [ アクション] [> 送信データの表示...] の順にクリックしますフォームデータが表示されます。
