    Salesforce用のカスタムリードフォームを作成する

    このトピックでは、Salesforce フォームを使用して Brightcove Player 内でリードデータをキャプチャする方法について説明します。

    オーディエンスモジュールでは、カスタムリードフォームを作成できます。これにより、Brightcove プレーヤー内で動画を再生するときに視聴者情報をキャプチャできます。リードデータは Salesforce に送信され、Salesforce によって処理されます。

    カスタムリードフォームを作成するには、次の手順を行います。

    1. Salesforce でフォームを作成する
    2. フォームをコピーして編集する HTML
    3. オーディエンスモジュールでカスタムリードフォームを作成する
    4. リードフォームをプレーヤーに割り当てる
    5. Salesforce でフォームデータを検証する
    注：視聴者の情報をキャプチャし、Video Cloud にデータを保存する標準的なリードフォームについては、「オーディエンスリードフォームの作成」を参照してください。

    Salesforce でのフォームの作成

    Salesforce の Webto-Lead 機能を使用すると、Web サイトの訪問者からリード情報を生成できます。これらのフォームは、ブライトコーブプレーヤーの内部に表示することもできます。以下の手順は、名、姓、メールアドレス、会社、都市、州/省を取得するための簡単なフォームを作成します。Webto-Lead 機能の詳細については、 Salesforce のドキュメントを参照してください。

    1. Salesforce アカウントにサインインします。
    2. [ 設定] から、[クイック検索] ボックスに」Web-to-Leadと入力し、[ Web-to-Lead ] をクリックします。
    3. [ Web-to-Lead フォームの作成 ] をクリックします。
    4. Web-to-Lead フォームに含めるフィールドを選択します。[追加] および [削除] 矢印を使用して [使用可能なフィールド] リストと [選択したフィールド] ボックスの間でフィールドを移動し、上向き矢印と下矢印を使用してフォーム上のフィールドの順序を変更します。
    5. 返品URLを入力します。この例では、返品 URL は http://www.brightcove.com に設定されます。この値は、フォームが Audience モジュール内でカスタム潜在顧客フォームとして使用される場合、無視されます。
    6. [ 生成 ] をクリックします。フォーム HTML が表示されます。

    フォームのコピーと編集 HTML

    Audienceモジュール内部でフォームを使用するには、フォームHTMLをコピーする必要があります。また、フォームに基本的なスタイリングを行うこともできます。以下の手順では、HTMLエディタにフォームHTMLをコピーし、フォームにいくつかのスタイル変更を行います。

    デフォルトでは、フォームは　Brightcove Player内部で表示されると、プレーヤー上の透明な背景に表示されます。背景色を設定するには、CSS を使用します。

    1. HTML をクリップボードにコピーします。
    2. [ 完了 ] をクリックします。
    3. フォームHTMLを、お気に入りのHTMLエディターに貼り付けます。エディターのコード整形機能を使用して、読みやすいようにコードを整形したい場合があります。
    4. HTML エディタを使用して、次の CSS を追加して、背景色を灰色に変更します。
      <style>
      body {
      background-color: rgba(204,204,204,0.9);
      margin: 0;
      padding: 20px;
      height: 100%;
      }
      </style>
    5. （オプション）オーディエンスが生成するリードを識別するために、<form>タグ内に次のコードを追加できます。
      <input type="hidden" name="lead_source" value="Video Cloud Audience">
    6. HTML をクリップボードにコピーします。

    Audienceモジュールでカスタムリードフォームを作成する

    Salesforce でフォーム HTML が生成されたら、オーディエンスモジュールでカスタムリードフォームを作成し、リードフォームを Audience 対応プレーヤーに関連付けます。

    1. Audienceモジュールを開きます。
    2. 左側のナビゲーションで [ リードフォーム ] をクリックします。リードフォームが作成されている場合、作成されたリードフォームが表示されます。
    3. [リードフォームの作成]をクリックします。
    4. リードフォームに名前を付けます。
    5. フォームの言語を選択します。
    6. フォームのタイミングを選択します。フォームのタイミングで、リードフォームがいつ表示されるかが決まります。
    7. フォームのタイプを[カスタムリードフォーム]に設定します。
    8. Salesforce からフォーム HTML に貼り付けます。
    9. （オプション）確認 HTML に貼り付けます。確認ビューが5秒間表示され、その後、動画の再生が再開されます。確認HTMLが含まれない場合は、フォームの送信後に動画の再生が再開されます。
    10. （オプション）フォームのプレビューを開くには、[フォームのプレビュー]をクリックします。
    11. [保存]をクリックして、リードフォームを保存します。

    リードフォームをプレーヤーに割り当てる

    作成したリードフォームは、プレーヤーに割り当てる必要があります。リードフォームをプレーヤーに割り当てるには、次の手順を行います。

    1. 左側のナビゲーションで [ オーディエンスプレイヤー ] をクリックします。
    2. プレーヤーを見つけ、[ Lead Form ] ドロップダウンリストを使用して、リードフォームを選択します。

    これでセットアップは完了しました。リード情報を取得する準備ができました。Audience対応プレーヤーを使用して動画が公開されると、フォームタイミング設定に従ってリードフォームが表示されます。

    Salesforce でのフォームデータの検証

    カスタムリードフォームを使用してキャプチャされたリードは、Salesforce アカウントのリードとして追加されます。
